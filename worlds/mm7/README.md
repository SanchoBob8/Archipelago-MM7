Mega Man 7 Archipelago World

Status: MVP / development release

Features:
- Robot Master reward randomization
- Rush item checks
- Rush plate checks (finding all 4 will grant the Super Adaptor, which is not randomized)
- Proto Man hints/shield checks
- Auto's Bolt/Energy Balancer/Exit Unit/Hyper Rocket Buster checks
- Freeing Beat check
- Giant Bolts and Health Capsule checks
- Wily access code progression
- Wily boss reward checks
- SNI client support

Requirements:
- Archipelago 0.6.7+
- Mega Man VII (USA) clean ROM
- SNES emulator with SNI support, such as BizHawk

Installation:
- Install mm7.apworld through Archipelago Launcher
- Generate using a Mega Man 7 YAML
- Open the generated .apmm7 patch through the Launcher
- Connect through SNI
