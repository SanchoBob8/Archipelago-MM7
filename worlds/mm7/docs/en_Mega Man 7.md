# Mega Man 7

## Where is the options page?

The [player options page for this game](../player-options) contains all the options you need to configure and export a config file.

## What does randomization do to this game?

Mega Man 7 randomizes Robot Master rewards, Rush items, Rush Plates, Proto Man checks, selected stage pickups, Wily Access Codes, and Wily boss rewards into the multiworld.

The game is completed by defeating Wily Capsule.

Access to Wily stages is controlled by Wily Access Codes. Wily 1, Wily 2, and Wily 3 each have their own access code. Wily 4 becomes available after clearing Wily 1, 2 and 3.

## What is considered a location check in Mega Man 7?

- Robot Master reward checks
- Proto Man meetings
- Proto Shield
- Rush item pickups
- Rush Plate pickups
- Auto's Bolt
- Exit Unit
- Hyper Rocket Buster
- Energy Balancer
- Beat rescue
- Giant Bolts
- Mega Health Capsule
- Intro stage clear
- Mash / Robot Museum clear
- Wily 1, 2 and 3 clear checks

Some items can be obtained either as stage pickups or by buying them in Auto's shop. Both methods point to the same Archipelago location, so the item can only be checked once.

## When the player receives an item, what happens?

The effects of the item are immediately applied in-game.

Weapons and Rush items become usable as soon as they are received. Rush Plates are tracked separately; collecting all four Rush Plates grants the Super Adapter. The Super Adapter itself is not a randomized item.

Consumable items such as 1-Ups and tanks are added to the player's inventory when received.

Wily Access Codes unlock access to their corresponding Wily stages from the stage select screen.

## What is the goal?

The goal is to defeat Wily Capsule.

Wily 4 becomes available after Wily 1, Wily 2, and Wily 3 have been cleared.

## What is different from vanilla Mega Man 7?

Robot Master rewards are AP-only. Defeating a Robot Master sends a location check instead of directly granting the vanilla weapon.

Some upgrades and pickups are also converted into Archipelago checks. When one of these checks is collected, the item placed there is sent through Archipelago instead of necessarily matching the original pickup.

The stage select screen allows selecting available Wily stages using the Wily stage box once Wily Access Codes have been received. You can cycle through the unlocked stages with L/R.

Mega Man 7 supports options for starting lives, bolts, E-Tanks, W-Tanks, and S-Tanks.

## Current limitations

This is an early MVP release. It should be fully playable, but options are currently limited.
